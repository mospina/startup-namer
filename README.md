## Flutter commands

Common flutter commands:

```
$ flutter create startup_namer
$ cd startup_namer/
$ flutter doctor
$ flutter devices
$ flutter run
```

## Emulator and screen

Running a emulator inside screen:

```
$ screen -S
$ cd ~/Android/Sdk/emulator
$ emulator -avd Pixel_2_API_29
$ CTRL a d
```
